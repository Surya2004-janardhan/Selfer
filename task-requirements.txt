Selfer 

selfer is an intellegent (calude code + open claw) like system but that can run on local ollama support or third party service providers,
it connects user Telegram bot for thrid part access to the current repo or it can also handle CLI based chat interface.

Selfer is global installed package vai pip install selfer where several commands are prebuilt to use in the CLI once selfer is installed 
and connected to a directory it has access to that directory completely you can give commands vai CLI or chat at CLI within the system 
or can send messages from your Telegram bot that you have connected earlier.

Once the session gets start selfer holds only repo/directory memory it doesnt hold any global memory, selfer checks its previous sessions if previous session is there
it goes directory to there continues your chat if not there is creates a directory memory where it comes to know all the things present in the
directory/repo

Selfer uses init command to start and understant the repo and creates its own .selfer directory that has essentail info needed for the system to handle the workplace

selfer is local first multi agent orchestration where it has buch of agents with its hands with strong system prompts and security checks
