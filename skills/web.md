# Web Systems & Protocols

## Networking
- **HTTP/1.1 vs HTTP/2/3**: Understanding multiplexing, header compression, and server push.
- **DNS**: Propagation, A/AAAA records, and CNAME flattening.
- **WebSocket/gRPC**: Real-time, bi-directional communication vs high-performance RPCs.

## Security
- **TLS/SSL**: Managing certificates and enforcing HTTPS.
- **CORS**: Correct configuration of cross-origin resource sharing policies.
- **CSP (Content Security Policy)**: Mitigating XSS by restricting source origins.

## Infrastructure
- **CDNs**: Global content delivery (Cloudflare, Akamai) for low-latency assets.
- **Load Balancing**: Distributing traffic across multiple server instances (Nginx, HAProxy).
- **Reverse Proxies**: SSL termination, caching, and request routing at the edge.

## Standards
- **Fetch API**: Modern asynchronous request handling.
- **Semantic Tags**: SEO and accessibility standard adherence.
